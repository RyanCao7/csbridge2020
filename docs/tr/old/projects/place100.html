100 Koymak (ve çözümü)

Problemi Yazan: Chris

Ders Notları: Karel Referansı
Day1: Place100.java

Hedefiniz Karel'ın başladığı yerin önündeki karede 100 beeper'dan oluşan bir yığın oluşturmasını sağlamak. Lütfen bir satırda 100 kez putBeeper yazmayın!!! (ipucu: for döngüsü kullanın.)

Çözüm

/**
 * Program: Place100
 * -------------
 * This program makes karel place a pile of 100 beepers. Good times.
 */
public class Place100 extends SuperKarel {
	
	public void run() {
		move();
		// This for-loop will repeat the code inside 100 times.
		for(int i = 0; i < 100; i++) {
			putBeeper();
		}
		move();
	}
}
