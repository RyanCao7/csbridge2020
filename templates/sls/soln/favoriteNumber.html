This is one section problem for the day. Here is information about the goals for the problem and what you should focus on during section:
Learning Goal: using variables with while and conditions
Milestone: play the game with no feedback - just guess until correct

Write code on the board. Try to not use pseudocode! The hardest part is that guess must be reassigned...and in general this section is hard, but not many lines of code. Do your best to pace yourself. Try not to have students with experience give it away.

Solution

public class FavoriteNumber extends ConsoleProgram {

	private static final int FAVORITE_NUMBER = 88;
	
	public void run() {
	    println("Try and guess my favorite number (between 0 and 100)");
	    
        int guess = readInt("Guess: ");
        while (guess != FAVORITE_NUMBER) {
            if (guess < FAVORITE_NUMBER) {
                println("Too low");
            } else if (guess > FAVORITE_NUMBER) {
                println("Too high");
            } 
            
            guess = readInt("Guess: ");
        }

        println("Good job!");
	}

}
