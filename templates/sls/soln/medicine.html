This is one section problem for the day. The main learning goal is using variables and concatenating different variable types together. Here is the main way to understand how to approach the problem:

  • Initially, we may think to just loop NUM_PACKAGES times and each time print out a random number, but we may print out the same random number more than once. We need a way to print out numbers that have not been printed out before.
  • One way to do this is to make just part of the number completely random. In other words, let's make the first five digits a fixed number that we increment each time we make a medicine number. Then, let's make the last five digits a random number between 0 and 99999.
  • This will make a number up to 10 digits long in total, where no two numbers will be the same. This is because the first 5 digits are always unique. This also makes a number that is random, since the last 5 digits are always random.

Solution

public class Medicine extends ConsoleProgram {
  
    // The number of medicine numbers to generate
    private static final int NUM_PACKAGES = 1000;

    private static RandomGenerator rg = new RandomGenerator();
  
    public void run() {
        int firstFiveDigits = 10000;
        for (int i = 0; i < NUM_PACKAGES; i++) {
            println("" + firstFiveDigits + rg.nextInt(0, 99999));
            firstFiveDigits++;
        }
    }

}
